# Chromium command-line switches

Chromium command-line switches change frequently, so Thorium does not maintain
a copied snapshot in this repository. Use the continuously updated
[Chromium Command Line Switches](https://peter.sh/experiments/chromium-command-line-switches/)
reference instead.

Thorium's small set of development-oriented launch flags remains available in
[`infra/DEV_CMDLINE_FLAGS.txt`](../infra/DEV_CMDLINE_FLAGS.txt). Some development
switches weaken browser security and should only be used with disposable test
profiles.
