# Thorium AVX-512

This directory contains build config files for compiling Thorium/Chromium with
the `avx512_skx` profile: AVX2, FMA, F16C, and the AVX-512 F/CD/VL/BW/DQ
subsets. It does not imply later AVX-512 extensions such as VNNI or IFMA.
